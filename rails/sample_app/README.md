This project is simple authentication app 

 Technologies:
* Ruby-version: 2.5.3
* Rails-version: 5.2.2
* Database: MySQL

 Setup:
* Run to create database and table
  $ rails db:create
  $ rails db:migrate

* Run to install libraries
  $ bundle install

 Features:
* Signup, login and save cookies, logout 
* Edit user profile 
* Delete user with admin role
